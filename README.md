<h1>Python - Password Generator </h1>

<h2>Description</h2>
This project consists of a Python script that generates a password using a randomized combination of letters, numbers, and special characters, with a length of 10 characters. This is a useful tool to create a password that is hard to brute-force.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Python 3</b> 

<h2>Environments Used </h2>

- <b>Windows 11</b> 
- <b>Visual Studio Code</b>
